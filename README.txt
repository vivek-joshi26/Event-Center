Instructions
The project has two folder namely, backend and frontend.
frontend-
   1. Install node if not installed.
   2. do npm install
   3. then, npm start
   * Please wait for 1-2 seconds to wait for API response.
backend-
   1. Install maven
   2. Use mvn package or mvn install command 
   3. Run DemoApplication class
